# Tovi-bash

Bash utility for tracking time spent on different tasks.

Enable task name autocompletion.
```bash
source tovi-completion.bash
```

Start a task by name.
```bash
./start-tovi myTask
```

Stop a task by name.
```bash
./stop-tovi myTask
```

Measure total time spent on a task.
```bash
./duration myTask
```
